# HistoVibe — Dating with Historical Museums

A Tinder-inspired mobile app that helps young people discover and engage 
with museum collections in a fun, interactive way.

## Problem
Young people find museums boring and hard to navigate — 
low engagement with cultural heritage among Gen Z.

## Solution
HistoVibe gamifies museum discovery with a swipe-based interface, 
letting users "match" with exhibits and get notified about museum events.

## My Role
- Conducted market research and competitive analysis
- Designed and ran user surveys & interviews to identify pain points
- Defined core app features based on user insights
- Built interactive prototype in Figma
- Prepared project budget and feasibility report

## Key Features (Prototype)
- 🔥 Swipe interface to explore museum exhibits
- 📅 Event notifications for museum activities  
- 🗺️ Museum navigation and exhibit discovery
- 👤 Personalized recommendation based on interests
